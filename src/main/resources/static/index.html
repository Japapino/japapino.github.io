Review Website

Review website of movies to show case the use of HTML, Spring, and Thymeleaf.

Fullstack review website for movies. Java back end Spring MVC and Thymeleaf is used to populate the page. HTML, Javascript, CSS used for styling. Using the side buttons, the layout can be rearranged alphabetically. Searching by tag or genre is also added.

Repo

Fund It

Fullstack financial planner

Boot camp capstone project. This is a full stack, single-page financial planning application. Using this, the user can use a single savings account to plan and save for multiple goals such as a vacation, education, etc. Users assign tags as funds to an account, with this they can allocation savings into different funds. Users can deposit into a savings and choose how much of the deposit goes into each fund. They can also view the transaction history of each fund to stay on track on meeting their financial goals.

Virtual Pet Simulator

This application is a virtual pet simulation similar to that of a Tomagachi. Pets can be cats, dogs, robotic cats, or robotic dogs. Robotic versions of pets do not require to eat or drink, but do require maintenance.
